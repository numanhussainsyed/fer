About This Project

EmotionAI is a web-based application that utilizes deep learning and computer vision to detect facial emotions in real-time. Built with TensorFlow, Keras, Flask, and OpenCV, it provides a user-friendly interface and real-time feedback.

Features include live webcam detection, emoji feedback, sound output, and emotion analytics.
